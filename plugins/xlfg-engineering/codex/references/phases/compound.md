# Compound Phase

## Objective

Promote only verified, reusable lessons into durable knowledge.

## Required Artifacts

Update `docs/xlfg/knowledge/` only when there is a durable lesson. Update the run summary when useful.

## Process

- Read `verification.md`, `review-summary.md` when present, and final `spec.md`.
- Capture patterns, failures, and decisions that will help a future run.
- Do not copy the whole run into durable memory.
- Use the existing ledger writer when appending a new durable ledger event.
- Leave a concise next-agent handoff when the repo state changed.

## Done Check

Durable knowledge changed only for verified reusable lessons, and the final run summary names what was compounded or why nothing was.
